### Требования:

- node 7.4.0

### Описание:

Реализация алгоритма Дейкстры для решения задачи Коммивояжера на языке JavaScript.

Реализация не подразумевает ввода со стороны пользователя. Начальные и конечные ноды вбиты в сам код.
